BadRapApi

Easy-to-use api to get some of the worst rap lyrics of all time. Send a simple GET request and get back JSON data. Simple and easy to use, unlimited requests.

Some song lyrics are explicit

List of artists

/api

This endpoint will randomly fetch a single bad rap lyric from database

Returned JSON structure:

  • lyrics - the bad lyric of the song (text)
  • artist - the rapper who said the lyrics (text)
  • song - the song in which the lyrics were said{" "} (text)
  • release_date - the date on which the song was released{" "} (text)
  • explicit - whether or not song contains cuss words{" "} (boolean)

/api/filter

This endpoint will fetch a random song that matches current query. Endpoint MUST contain at least one query.

Valid queries

  • ?artist

    • This will filter requests by artist
    • Artist query must be all lowercase with no special characters
      • Dr. Dre = drdre
      • Jay-Z = jayz
      • Andre 3000 = andre3000
    • example: /api/filter?artist=jayz

  • ?explicit

    • Returns songs that either have explicit language or not
    • Can only be true or false
    • example: /api/filter?explicit=false

  • ?year

    • Will filter out songs by year it came out
    • value is in year format, do not need month or day
    • example: /api/filter?year=1985
